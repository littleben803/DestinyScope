# 八字命镜隐私政策

适用于八字命镜当前版本。英文/内部品牌名：DestinyScope。

最后更新：2026-06-11

八字命镜是一款完全 Native、本地运行的东方命理自我探索与传统文化学习 App。当前功能以设备端本地处理为基础，不创建账号，不接服务端，不接在线生成服务。

## 中文隐私政策

### 1. 当前正式功能

八字命镜当前正式功能不创建账号，也不需要登录。应用内没有用户注册、身份认证、云同步或个人资料系统。当前正式功能不接服务端，不包含付费订阅，不接入广告追踪，也不接入分析 SDK。

### 2. 出生信息处理

你输入的出生日期和出生时辰仅用于设备端本地传统命理计算。八字命镜当前版本不上传出生信息，也不会上传命理结果。

### 3. 本地历史记录

历史记录仅保存在本地设备，用于在 App 内查看过去生成的轻量结果。当前版本不上传历史记录，也不进行云同步。

### 4. 知识库收藏与最近阅读

如果你收藏知识库文章或浏览文章详情，当前版本只会在当前设备保存文章标识和阅读时间，用于展示收藏和最近阅读。当前版本不保存知识库文章全文，不保存搜索关键词，不上传、不同步、不需要账号。你可以在知识库页面取消收藏、清空收藏或清空最近阅读。

### 5. 本地数据管理

你可以在设置中的本地数据管理页面清理历史记录、知识库收藏、最近阅读和首次使用说明状态等本机数据。删除仅影响当前设备，不影响 App 内置知识库、称骨数据或本地计算规则。当前版本无账号、无同步、无服务端。

### 6. 系统权限

八字命镜当前版本不使用定位、相机、相册、通讯录、麦克风等敏感系统权限，也不会主动请求这些权限。

### 7. 广告、追踪与分析

八字命镜当前版本不接入广告追踪，不接入分析 SDK，也不包含跨应用追踪能力。

### 8. 付费订阅

八字命镜当前版本不包含付费订阅、内购或支付功能。

### 9. 系统分享

当前版本如调用 iOS 系统分享面板，仅在你主动点击分享时把当前结果摘要交给系统分享能力处理。八字命镜不会自动分享内容，不会从其他 App 接收分享数据，也不会通过分享功能上传出生信息、命理结果或历史记录。

### 10. 本地资源

本地命理数据来自 App Bundle 内置 CSV。本地知识库来自 App Bundle 内置 JSON 和 SQLite 预留资源。应用在设备端读取这些资源完成展示和计算。

### 11. 未来版本

如果未来版本加入联网、账号、订阅、数据同步、分析 SDK 或其他需要处理用户数据的能力，我们会在对应版本中更新隐私政策，并在必要时补充 App 内说明。

### 12. 联系方式

如对本隐私政策有任何问题，请联系：littleben803@gmail.com

## English Summary

This privacy policy applies to the current version of DestinyScope. DestinyScope is a fully native, local-first iOS app for Eastern metaphysics-inspired self-exploration and traditional culture learning.

- No account is created and no login is required.
- Birth date and birth hour are used only for on-device calculation.
- Knowledge favorites and recent reads, when used, store only article IDs and reading time on the current device.
- Local data management only deletes user data on the current device and does not remove bundled knowledge articles or local calculation rules.
- DestinyScope does not upload birth information, fortune results, or local history records.
- DestinyScope does not use a backend server for the current official features.
- DestinyScope does not request Location, Camera, Photos, Contacts, Microphone, or other sensitive permissions.
- DestinyScope does not include ad tracking, analytics SDKs, subscriptions, or in-app purchases.
- Fortune data is bundled as local CSV files, and knowledge resources are bundled as local JSON files.
- System sharing, when used, is initiated by the user through the iOS share sheet and does not upload data through DestinyScope.
- If future versions add networking, accounts, subscriptions, or data sync, this privacy policy will be updated.
