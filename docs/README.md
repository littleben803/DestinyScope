# DestinyScope GitHub Pages

This directory contains static public pages for DestinyScope.

## Privacy Policy

- Source Markdown: `privacy/privacy.md`
- Static page: `privacy/index.html`
- Expected GitHub Pages URL: `https://littleben803.github.io/DestinyScope/privacy/`

The privacy page is a standalone static HTML file. It does not require npm, Jekyll, external JavaScript, external CSS, remote fonts, or backend services.

## Support

- Static page: `support/index.html`
- Expected GitHub Pages URL: `https://littleben803.github.io/DestinyScope/support/`

The support page is a standalone static HTML file for the App Store Connect Support URL field. It mirrors the app's About positioning and provides a public contact path.
