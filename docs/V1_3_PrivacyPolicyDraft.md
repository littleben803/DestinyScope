# DestinyScope V1.3 Privacy Policy Draft

更新日期：2026-05-29

## 1. 当前 Release 隐私状态

当前 Release 功能仍是本地计算和本地模板生成：

- 无账号。
- 无登录。
- 无服务端。
- 无在线 AI。
- 无广告追踪。
- 无支付。
- 无定位权限。
- 无相机权限。
- 无相册权限。
- 无通讯录权限。
- 无麦克风权限。
- 出生日期和出生时辰仅用于设备端本地计算。
- 历史记录仅保存在本地设备。
- 本地知识库和称骨数据来自 App Bundle 内置资源。

当前 Release 不应在隐私政策中宣称已开放生产级本地模型能力。

## 2. 未来开放本地模型润色时需要补充

如果未来在 TestFlight 或生产功能中开放本地模型润色，需要补充：

- 本地模型仅在设备端运行。
- 模型输入来自已有模板文本和结构化命理摘要。
- 不上传出生信息。
- 不上传命理结果。
- 不上传模型输入。
- 不上传模型输出。
- 不上传历史记录。
- 不使用云端 AI。
- 不使用第三方在线模型服务。
- 模型失败、安全检查失败或超时时，会回退本地模板文本。
- 本地模型只做文本润色，不生成新的命理结论。

## 3. 未来支持模型下载时需要额外补充

如果未来支持模型下载，需要额外说明：

- App 会请求网络下载模型文件。
- 模型文件大小。
- 模型文件存储位置。
- 用户是否可以删除模型文件。
- 下载模型不会上传用户出生信息、命理结果、历史记录或模型输入输出。
- 下载完成后会使用 sha256 或等价方式校验文件完整性。
- 下载服务或 CDN 来源。
- 下载失败、存储空间不足和校验失败的处理方式。

当前 V1.3 不设计正式模型下载功能。

## 4. 历史记录说明

- 历史记录仍只保存在本地设备。
- 本地模型润色结果默认不写入历史记录。
- 本地模型润色结果默认不覆盖原始命理结果。
- 如果未来允许保存润色结果，需要用户确认。
- 如果未来允许保存润色结果，需要提供删除能力。
- 如果未来允许同步或导出润色结果，需要单独更新隐私政策。

## 5. 推荐隐私政策草案文案

### 当前 Release 可用文案

```text
DestinyScope 当前版本不创建账号，不需要登录，也不接入服务端或在线 AI。用户输入的出生日期和出生时辰仅用于设备端本地命理计算，不会上传到服务器。

App 内的称骨数据、知识库和模板解读来自 App Bundle 内置资源。历史记录仅保存在本地设备，当前版本不进行云同步。

当前版本不使用定位、相机、相册、通讯录、麦克风等敏感权限，不包含广告追踪、支付订阅或第三方分析 SDK。
```

### TestFlight 本地模型实验文案

```text
在 TestFlight 内测中，DestinyScope 可能提供“本地模型润色实验”。该功能仅使用设备端本地模型，对已经生成的模板文本和结构化命理摘要做表达润色。

本地模型实验不会上传出生信息、命理结果、模型输入、模型输出或历史记录。该功能不使用云端 AI，也不使用第三方在线模型服务。

本地模型只用于文本润色，不生成新的命理结论，不替代称骨计算、农历转换、诗文匹配或模板解释。模型失败、安全检查失败或超时时，App 会回退本地模板文本。
```

### 未来模型下载文案占位

```text
如果未来版本支持模型文件下载，App 会通过网络下载模型文件，并在设备端校验文件完整性。下载模型不会上传用户出生信息、命理结果、模型输入、模型输出或历史记录。

模型文件将保存在设备本地，用户可在 App 提供的入口中查看或删除。具体下载来源、文件大小、校验方式和存储位置会在正式功能上线前补充说明。
```

该段仅作为未来占位，不应在当前 Release 隐私政策中启用。
