# DestinyScope V1.1 知识库资料来源

更新日期：2026-05-27

本文件记录 V1.1 知识库扩充可参考的资料来源。所有 App 内文章必须原创改写，不能复制来源不明的大段文字，不能采用强预测、恐吓、改命、化解、避灾类文案。

## 使用原则

- 优先使用公开、可引用、公版或机构类资料作为 reference。
- 古籍原文只作为概念、术语和传统文本背景参考，App 文章需要用现代通俗语言重新组织。
- 机构类民俗文章可作为生肖、地支、时辰等文化背景参考。
- 百科类资料只用于基础概念核对，不作为大段正文来源。
- 民间称骨资料站、普通算命网站、PDF、Scribd 文档等只能谨慎了解流传形态，不直接复制，不采用强断言。

## 禁止内容边界

知识库文章和未来 RAG 资料不得写入或暗示：

- 精准预测。
- 改命。
- 化解。
- 避灾。
- 必然发财。
- 寿命疾病预测。
- 投资确定性。
- 婚姻确定性。
- 恐吓式大凶判断。
- 引导购买服务、法物、课程或咨询。

推荐使用表达：

- 传统文化参考。
- 自我探索。
- 可以关注。
- 倾向。
- 建议。
- 观察角度。
- 入门理解。
- 不作为现实重大决策依据。
