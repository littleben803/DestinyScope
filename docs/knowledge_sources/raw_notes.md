# DestinyScope V1.1 知识库原始整理笔记

更新日期：2026-05-27

本文件用于记录资料整理要点。这里不是 App 正文，不直接写入 `knowledge_articles.json`。后续 V1.1 阶段 4B 扩充 JSON 时，需要基于这些笔记进行原创改写。

## 总体整理原则

- 每篇 App 文章控制在 300 到 600 字。
- 使用通俗现代语言解释传统概念。
- 不复制古籍、网页或 PDF 的大段原文。
- 可以少量引用术语、卦名、概念名，但正文必须重新组织。
- 所有内容保持“文化学习、自我观察、参考性解释”定位。
- 不写强预测，不写恐吓，不写改命、化解、避灾。

## 称骨算命

可整理方向：

- 称骨算命是一种以出生年月日时折算重量，再对应诗文的民间命理方法。
- App V1 的重点是展示权重明细、总重量、诗文和参考性解读。
- 称骨资料的现代流传版本较多，来源质量参差不齐。
- 民间称骨资料站、PDF、Scribd、普通算命网站只能用于了解流传形态，不作为权威正文来源。
- 文章中需要强调结果不可按字面绝对理解，诗文需要结合现代语境解释。

可拆文章：

- 称骨算命是什么。
- 称骨算命的计算方式。
- 称骨结果如何理性看待。
- 称骨诗文为什么需要解释。

## 农历、时辰、天干地支

可整理方向：

- 农历是中国传统历法系统，常与节气、月份、传统节日和民俗活动相关。
- 十二时辰以地支表示一天的时间段，是传统时间表达方式之一。
- 天干和地支可组成六十甲子，用于纪年、纪月、纪日、纪时等传统表达。
- 生肖与地支存在对应关系，生肖文化也包含民俗、传说和日常象征。
- CCTV 和中国文艺网资料可用于生肖源流、时辰传说、民俗背景说明。

可拆文章：

- 农历基础。
- 十二时辰。
- 天干基础。
- 地支基础。
- 天干地支组合。
- 六十甲子入门。
- 十二生肖基础。
- 生肖与地支的关系。

## 五行

可整理方向：

- 五行是木、火、土、金、水五类传统象征概念。
- 五行常用于描述变化、关系、节奏和平衡，不是现代科学意义上的物质分类。
- 相生可解释为一种支持、转化或延展关系。
- 相克可解释为一种制约、边界或平衡关系。
- CText 五行生克片段可用于核对传统生克序列。
- 百科类资料只用于概念核对，正文需原创改写。

可拆文章：

- 五行基础。
- 五行相生。
- 五行相克。
- 五行与自我观察。

## 八卦与周易

可整理方向：

- 《周易》和八卦适合作为传统文化入门内容，重点解释象征和思维方式。
- CText《周易》和维基文库《周易》可用于卦名、传统文本和分卦原文参考。
- 八卦文章应避免做个人命运断言，只做象征含义和自我观察角度说明。
- 八个单卦可分别解释为观察角度：刚健、承载、启动、入微、风险、表达、停止、交流。

可拆文章：

- 八卦基础。
- 乾卦：刚健与自强。
- 坤卦：承载与包容。
- 震卦：行动与启动。
- 巽卦：入微与沟通。
- 坎卦：风险与深思。
- 离卦：光明与表达。
- 艮卦：停止与边界。
- 兑卦：悦纳与交流。
- 周易入门。

## 梅花易数

可整理方向：

- 维基文库《梅花易数》可作为传统文本框架参考。
- V1.1 只做入门介绍，不实现梅花易数起卦或断卦功能。
- 文章应强调其作为传统象数思维的一种代表，不做具体预测承诺。

可拆文章：

- 梅花易数入门。

## 自我探索与使用边界

可整理方向：

- 命理内容可作为传统文化阅读和自我观察工具。
- 用户不应仅依据 App 结果做现实重大决策。
- 需要明确不构成医疗、法律、财务、投资、婚恋、职业等现实决策建议。
- 需要避免精准预测、改命、化解、避灾、必然发财、寿命疾病预测、投资确定性、婚姻确定性等表达。

可拆文章：

- 命理与自我探索。
- 命理结果的使用边界。

## 后续 4B 写入 JSON 前检查

- 每篇文章是否原创改写。
- 每篇文章是否 300 到 600 字。
- 是否有 `id`、`category`、`title`、`summary`、`body`、`tags`、`source`、`version`。
- 是否避免复制来源原文。
- 是否避免高风险现实决策承诺。
- 是否保留“仅供娱乐、自我探索和传统文化学习参考”的边界。
