# DestinyScope App Store 元数据草案

更新日期：2026-05-27

本文件用于准备 App Store Connect 上架文字材料。所有文案应保持 V1 边界：传统文化、自我探索、本地处理、无账号、无服务端、无在线 AI。

## 1. App 名称建议

首选：

```text
DestinyScope
```

备选：

```text
DestinyScope 命镜
```

建议优先使用英文品牌名 `DestinyScope`，保持国际化空间，并避免中文名过度强化“算命”单一属性。

## 2. 副标题建议

首选：

```text
东方命理与本地命盘解读
```

备选：

```text
称骨命理与传统文化入门
```

```text
出生时辰的本地文化解读
```

说明：副标题避免使用“精准预测”“改命”“大师”等高风险表述，重点放在东方命理、传统文化和本地解读。

## 3. Promotional Text

```text
DestinyScope 以本地称骨命理数据和传统文化知识库为基础，帮助你从出生日期与时辰出发，获得参考性的命格诗文、五类解读与东方命理入门内容。无需账号，无服务端，出生信息仅在设备端处理。
```

## 4. Description

```text
DestinyScope 是一款完全 Native、本地运行的东方命理自我探索工具。你可以输入出生日期与出生时辰，App 会基于内置称骨命理数据，在设备端生成称骨结果、命格诗文、年/月/日/时权重明细，以及总评、性格、事业、财运、关系五类参考性解读。

我们希望 DestinyScope 不只是一个简单的测试工具。V1 版本加入了本地知识库，帮助你了解称骨算命、农历时辰、生肖、五行和命理结果的使用边界。你可以把它作为了解东方传统文化和自我探索的一种轻量入口。

隐私与离线体验：
- 无需创建账号
- 无需登录
- V1 不接服务端
- V1 不接在线 AI
- 出生日期和时辰仅在设备端用于本地计算
- 本地知识库和命理数据来自 App 内置资源
- 不使用定位、相机、相册、通讯录、麦克风等权限

内容边界：
DestinyScope 的结果仅供娱乐、自我探索和传统文化学习参考，不构成医疗、法律、财务、投资、婚恋或职业决策建议。请勿仅依据 App 结果做重大现实决策。
```

## 6. 分类建议

首选分类：

```text
Lifestyle
```

备选分类：

```text
Reference
```

选择建议：

- 如果更强调自我探索、轻量测试和日常使用，选择 Lifestyle。
- 如果更强调传统文化知识库、术语学习和内容阅读，选择 Reference。
- V1 当前同时具备结果页和知识库，建议首选 Lifestyle，副分类可考虑 Reference。
